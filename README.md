# Gauge-Designer-Studio-GDS-
A Python-based program for Windows that can help you design your own gauge in a simple and fun way

This app was born out of necessity: the need to use my own designs in my driving sim's custom tachometer. I tried other generic drawing software, but it was difficult, imprecise, and frustrating. So I decided to share my work with others as well. Whether designing sporty custom graphics or simply reproducing existing ones for repairs, the results came out as perfect replicas.

## Main features
- **Project tab:** The main tab where you can save and load your design, undo and redo changes, change the language (currently English and Italian only), swap the unit with a ratio (pixels or centimeters), and adjust final export settings and formats.

- **Wallpaper tab:** Here you can change the wallpaper style, color, overall size, enable the outer bezel, set the minimum and maximum scale values, and define the start and end angles of the scale.

- **Indicator tab:** Use this to customize the look of the entire needle (split into four parts: point, hub, decor ring, counterweight), including color, shape, and smoothing when simulating values. *Data Simulation Binding* lets you test your design in some driving games. ***For now only BeamNG.drive*** is supported, but I'd like to make it compatible with every game that supports UDP.

- **Typography tab:** Adjust the appearance of all text and number elements.

- **Ticks tab:** Ticks are divided into three categories: major, minor, and sub‑minor. Each category is fully customizable under its own sub‑tab.

- **Arcs tab:** Create fully customizable arcs for highlighting zones (e.g., redline).

- **Sections tab:** This ***overrides*** your current tick styling within a defined range. Also fully customizable.

- **Simulation tab:** Manually move the needle or set up a UDP connection. There's also an option to keep the program always on top.

- **Lock Dashboard Resize button:** Did you know the settings panel is resizable? If that bothers you, this button prevents resizing.

## Minimum system requirements:
- Electricity :)

Jokes aside, this can run really easily on any pc that has at least a little of GPU power. Tested (at full potential with really complex design) on `i5-7200U`, `HD Graphics 620` at `1920x1080` and it works like a charm!

## Bugs
- *Always on top* flashes when enabled/disabled, but works perfectly.
- I recommend to first choose if to use centimeters or pixels and then start designing.
- First install fonts, then you can open the app.
- UDP simulation is a work in progress.
